# Quiz Engine (C++)

A simple console-based quiz application written in C++.

## Features

- Multiple categories:
  - Geography
  - History
  - Programming
  - General Knowledge
- Case-insensitive answer checking
- Score tracking
- Final score displayed at the end of the quiz
- Input validation (blank answers handled)

## How It Works

1. User selects a category.
2. Questions are displayed one by one.
3. User enters answers.
4. The program checks answers ignoring case sensitivity.
5. Final score is shown after completing the quiz.

## Technologies Used

- C++
- STL (vector, string)
- Basic control structures (switch, loops, functions)
